# Repositorio de Add-ons Cartera Filios

Este repositorio contiene el addon **Cartera TEST** para Home Assistant.

## Instalación en Home Assistant

1. **Crea un repositorio en GitHub** con el contenido de esta carpeta:
   - Crea un repo nuevo (ej. `ha-cartera-addons`)
   - Sube `repository.yaml`, la carpeta `cartera_test` y su contenido

2. **Edita `repository.yaml`** antes de subir:
   - Cambia `TU-USUARIO` por tu usuario de GitHub
   - Cambia `ha-cartera-addons` por el nombre de tu repo (si es diferente)
   - Cambia `Tu Nombre` y `tu@email.com` por tus datos

3. **Añade el repositorio en Home Assistant:**
   - **Configuración** → **Complementos** → **Add-ons**
   - Clic en los **tres puntos** (⋮) → **Repositorios**
   - Añade: `https://github.com/TU-USUARIO/ha-cartera-addons`
   - Clic en **Añadir**

4. **Instala el addon:**
   - Busca **"Cartera TEST"** en la lista
   - Clic en **Instalar**
   - Inicia el addon

5. **Accede:** `http://TU-IP:8502`
